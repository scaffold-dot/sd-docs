## What is Scaffold-DOT?

Scaffold-DOT is an open-source toolkit for building decentralized applications on Polkadot Hub blockchain with Solidity smart contracts. It's a fork of the popular Scaffold-ETH 2, adapted for Polkadot's Asset Hub using the PolkaVM execution environment.

## Tech Stack

Built with modern web3 technologies:

- **Frontend**: NextJS, Reown AppKit, Wagmi, Viem, TypeScript
- **Smart Contracts**: Hardhat, Solidity, PolkaVM
- **Blockchain**: Polkadot Asset Hub with PolkaVM execution
